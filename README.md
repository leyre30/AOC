#Entrega final PRA - Advent of Code 2025

En este repositorio presentamos una selección de problemas de Advent of Code que hemos utilizado como portfolio para la asignatura PRA.
No se incluyen todos los días del calendario, sino solo aquellos que consideramos más representativos, tanto por el uso de estructuras de datos vistas en clase como por el nivel de razonamiento necesario para resolverlos.

#Día 02: implementación desde cero de una tabla hash para memorizar identificadores ya evaluados y evitar cálculos repetidos.

#Día 05: uso de un árbol binario de búsqueda propio para trabajar con intervalos y fusionar solapes.

#Día 7: simulación del recorrido de un rayo usando una cola BFS y una matriz de visitados para controlar ramificaciones y evitar ciclos.

#Día 11: grafo dirigido con conteo de caminos, incorporando restricciones adicionales en la segunda parte del problema.

Cada problema se encuentra en su carpeta correspondiente (diaXX/) e incluye el código fuente, el fichero de entrada y un pequeño README explicando el enfoque seguido.
Hemos priorizado soluciones claras y bien estructuradas frente a resolver un mayor número de problemas sin aportar nuevas ideas.
